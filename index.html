III. Lộ Trình Học AI/ML (6 Tháng - 24 Tuần, 600 Giờ)

Giai Đoạn 1: Nền Tảng Cơ Bản (4 Tuần, 100 Giờ)

  • Tuần 1-2: Python for Everybody (50 giờ)
    • 3-4 giờ/ngày: Cú pháp, cấu trúc dữ liệu, xử lý file
    • Thực hành: Script xử lý CSV
  • Tuần 3-4: Mathematics for Machine Learning: Linear Algebra (50 giờ)
    • 3-4 giờ/ngày: Vector, ma trận, bài tập

Giai Đoạn 2: Kiến Thức Cốt Lõi ML (8 Tuần, 200 Giờ)

  • Tuần 5-8: Machine Learning (Stanford - Andrew Ng) (100 giờ)
    • 3-4 giờ/ngày: Linear/Logistic Regression, Neural Networks
    • Thực hành: Dự đoán Titanic (Kaggle)
  • Tuần 9-12: Data Analysis with Python (IBM) (100 giờ)
    • 3-4 giờ/ngày: Pandas, NumPy, visualization
    • Thực hành: Phân tích dataset Kaggle

Giai Đoạn 3: Deep Learning & Ứng Dụng (8 Tuần, 200 Giờ)

  • Tuần 13-16: Deep Learning Specialization (Khóa 1 & 4) (100 giờ)
    • 3-4 giờ/ngày: Neural Networks, CNN
    • Thực hành: Phân loại ảnh (mèo vs chó)
  • Tuần 17-20: Machine Learning Engineering for Production (MLOps) (Khóa 1) (100 giờ)
    • 3-4 giờ/ngày: Triển khai Flask
    • Thực hành: Đưa mô hình lên web

Giai Đoạn 4: Chuẩn Bị Xin Việc (4 Tuần, 100 Giờ)

  • Tuần 21-24:
    • 3-4 giờ/ngày:
      • Ôn lý thuyết (1 giờ)
